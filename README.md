# Elloran Client

Frontend client code for *Dragons of Elloran Sea*.

This repository hosts all CSS and JavaScript used by the Forumotion-based game UI,
served via jsDelivr CDN.

## Structure
- /css — shared and page-specific styles
- /js  — core utilities and page logic

## Usage
Loaded directly into Forumotion pages via:
https://cdn.jsdelivr.net/gh/Draxion2/elloran-client@main/

No build tools required.
